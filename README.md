# Example of a React Native application using Redux, Redux toolkit and Redux Persist

# Getting Started

## Install modules:

```npm install```

### Install for iOS:

```
cd ios
pod install
```

## Start

```bash
# using npm
npm start

# OR using Yarn
yarn start
```
